宿舍留宿外人暨異性互訪規則

九十三年五月十三日 第十八次學生事務會議修正通過

一、 依據本校大學部宿舍管理要點第十條 、研究生宿舍管理要點第九條規定，為維護宿舍安全與安寧，特訂定本規則。

二、 本規則所稱之「留宿」，謂晚上十二時至凌晨五時仍留滯於宿舍者。

三、 住宿學生因特殊理由，至生活事務組專案申請核准，方得留宿外人。

四、 異性互訪規則如下：

  1. 異性可互訪時間：每日下午六點至十點。
  2. 欲進入異性宿舍，應先辦理換證手續後，並穿上訪客背心方可進入。
  3. 進入異性宿舍，受訪者必須陪同訪客，若無人陪同，則不得進入。
  4. 進入異性宿舍，其活動範圍限於宿舍走廊及宿舍交誼廳；若需進入異性寢室者，必須先徵得室友之書面同意並將同意書繳交管理員後方可進入。
  5. 非異性互訪時間不得進入宿舍，有特殊理由者須至生活事務組申請同意後方可進入。
  6. 進入異性宿舍，不得使用浴室。

五、 下列行為予以違規記點一次，由管理員口頭告誡，並填寫違規紀錄單：

  1. 互訪時間，未經換證即私自進入異性宿舍，訪客違規記點一次。
  2. 受訪者於訪客進入異性宿舍期間並未陪同訪客，受訪者及訪客均記點一次。
  3. 訪客未依規定穿著背心，受訪者及訪客均記點一次。
  4. 超過互訪時間經第一次通知未離開，受訪者及訪客均記點一次。

六、 下列行為予以違規記點二次，由生活事務組填寫書面警告單：

  1. 未取得受訪者室友同意書即擅自進入異性寢室者，受訪者及訪客均記點二次。
  2. 非互訪時間未經同意私自進入異性宿舍者，訪客記點二次，惟受訪者未盡義務立即提醒訪客辦理申請，受訪者亦違規記點二次。
  3. 進入異性宿舍使用浴室者，受訪者及訪客均記點二次。
  4. 超過互訪時間經第二次通知未離開，受訪者及訪客均記點二次。
  5. 違規記點累計滿兩次者。

七、 下列行為應勒令退宿或取消住宿資格：

  1. 私自留宿外人經查證屬實者，訪客暨受訪者皆予以勒令退宿；該寢在場之住宿同學視情節輕重予以記點兩次或取消住宿資格。
  2. 超過互訪時間經第三次通知未離開，受訪者及訪客均記點三次。
  3. 違規記點累計滿三次(含以上)者。
  4. 其他違規行為情節嚴重者。

八、 本規則經學生事務會議審核通過陳請校長核定後發怖實施，修正時亦同。
